Hotkeys

List of hotkeys available:

  1. Right Mouse Click on any input field (color, font, or text input) - the field will revert to default setting
  2. Ctrl + Shift + Left to open the bookmarks sidebar (if it is enabled in the settings)
  3. Ctrl + Shift + Right to open the main sidebar
  4. Ctrl + Shift + Up to create a sticky note (if it is enabled in the settings)
  5. Ctrl + V to create a sticky note with content from your clipboard (if it is enabled in the settings)
  6. Click on the Customization header below to collapse all sections in this sidebar
